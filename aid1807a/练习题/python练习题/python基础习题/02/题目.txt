练习2:
  输入三行文字，让这三行文字在一个方框内居中显示
  如输入(不要输入中文)：
  hello tarena!
  my name is weimingzi
  good bye

  |     hello tarena      |
  |  my name is weimingze |
  |      good bye         |
  +-----------------------+
